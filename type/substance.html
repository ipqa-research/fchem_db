type, public :: Substance

Substance object.

A Substance object holds all the defined properties available in the package. It can be initialized either by hand or from a json file. To initialize from a json file the user must provide a database path by setting up the variable fchem_db_dir. Then defining a Substance by it’s name (which should be the same as the filename) will find all the properties and set them up properly.

use fchem_db, only: Substance, fchem_db_dir
type(Substance) :: sus

fchem_db_dir = "some/directory/path"
sus = Substance("1-butanol")

print *, sus%critical%critical_temperature%value
